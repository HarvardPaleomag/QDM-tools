picking_tools¶

picking_tools.get_mask_extent(mask)¶

Takes a mask and retuns the extent of the smallest possible rectangle around it, where values ~= 0.

picking_tools.limit_mask(mask, kwargs)¶

Takes a mask and retuns the smallest possible rectangle where values ~= 0.

picking_tools.pick_area(data)¶

Function lets you pick several areas of an image and returns N masks, one for each area. The Mask is an array with the same dimensiona as the original image.

Returns

cell of n logical arrays one for each mask

picking_tools.pick_box(data, varargin)¶

pick_box lets you pick the ceter of an image and a box around it

positional parameters

data: QDM/LED data

optional positional parameters

n: int number of rectangles to pick

optional parameters:
led: bool

if true: colorscale ‘bone’ will be used for plotting

close_fig: bool

if true: figure will be closed after pick

Returns: cell

a cell with n entries. Each consisting of [x1,x2,y1,y2], where (x1,x2)
